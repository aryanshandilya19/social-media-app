# 🚀 Social Media App (Flutter + Node.js)

A full-stack social media application built using Flutter and Node.js with real-world features like authentication, feed, likes, comments, and follow system.

---

## 🔥 Features

* 🔐 JWT Authentication (Login / Signup)
* 📝 Create & Delete Posts
* ❤️ Like / Unlike Posts
* 💬 Comment System (Add / Delete)
* 👤 User Profiles
* ➕ Follow / Unfollow Users
* 🔍 Discover Users
* 📜 Infinite Scroll Feed

---

## 🛠 Tech Stack

**Frontend**

* Flutter (Dart)

**Backend**

* Node.js
* Express.js

**Database**

* MongoDB Atlas

---

## ⚡ Key Highlights

* RESTful API architecture
* Secure authentication with JWT
* Social graph (followers/following system)
* Optimized feed with pagination
* Clean and scalable folder structure

---

## 📂 Project Structure

```
backend/   → Node.js REST API  
frontend/  → Flutter Application  
```

---

## 📦 Setup Instructions

### 🔹 Backend

```bash
cd backend
npm install
npm run dev
```

---

### 🔹 Frontend

```bash
cd frontend
flutter pub get
flutter run
```
